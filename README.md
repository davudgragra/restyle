ReStyle
Мобильное приложение для покупки и продажи вещей в режиме реального времени.
Описание
ReStyle - это платформа для перепродажи одежды и других предметов гардероба. Пользователи могут выкладывать свои вещи на продажу, просматривать товары других пользователей, общаться с продавцами в чате, добавлять товары в корзину и оформлять заказы.
Основные функции
Для обычных пользователей:
⦁	Регистрация и авторизация
⦁	Просмотр ленты товаров
⦁	Поиск товаров
⦁	Добавление товаров в корзину
⦁	Оформление заказов
⦁	Чат с продавцами
⦁	Создание собственных объявлений
⦁	Личный кабинет с настройками профиля
Для администраторов:
⦁	Вход в админ-панель (email: admin@example.com, пароль: admin123)
⦁	Управление пользователями (блокировка/разблокировка/удаление)
⦁	Управление товарами (удаление)
⦁	Просмотр статистики (общее количество пользователей и товаров)
⦁	Поиск пользователей и товаров
Технические характеристики
⦁	Язык программирования: Java/Kotlin
⦁	Минимальная версия Android: 7.0 (API level 24)
⦁	Целевая версия Android: 14 (API level 34)
⦁	Архитектура: Фрагменты (Fragments) с нижней навигацией
Установка и запуск
1.	Клонируйте репозиторий или скачайте проект
2.	Откройте проект в Android Studio
3.	Дождитесь завершения синхронизации Gradle
4.	Подключите устройство Android или запустите эмулятор
5.	Нажмите кнопку "Run" в Android Studio
Структура приложения
Приложение состоит из следующих основных экранов:
1.	Лента товаров - главный экран с отображением всех доступных товаров
2.	Корзина - список товаров, добавленных в корзину с возможностью оформления заказа
3.	Добавление товара - форма для создания нового объявления
4.	Профиль - личный кабинет пользователя с настройками
5.	Заказы - история заказов пользователя
6.	Чат - общение с другими пользователями
Используемые библиотеки
⦁	AndroidX Core и AppCompat
⦁	Material Design Components
⦁	Glide/Picasso для загрузки изображений
